## Run Locally

Clone the project

```bash
  git clone https://github.com/IkuzaDev/TikViews
```

Go to the project directory

```bash
  cd TikViews
```

Install dependencies

```bash
  pip3 install -r requirements.txt
```

Install rich module

```bash
  pip3 install rich
```

Install halo module

```bash
  pip3 install halo
```

Start the bot

```bash
  python3 main.py
```

## Update from previous version

Pull the latest version

```bash
  git pull
```

Install new dependencies

``` bash
  pip3 install -r requirements.txt
```

Run the bot

```bash
  python3 main.py
```
